3. Access PatchMaster:

AGENT DEPLOYMENT:
-----------------
On agent machine (agent@agent):

1. Download agent package:
   curl -fsSL -o agent-latest.deb http://172.24.1.254:3000/download/agent-latest.deb

2. Install agent:
   sudo dpkg -i agent-latest.deb

3. Configure controller URL:
   echo 'CONTROLLER_URL=http://172.24.1.254:8000' | sudo tee /etc/patch-agent/env

4. Restart services:
   sudo systemctl restart patch-agent patch-agent-heartbeat

5. Verify agent is running:
   sudo systemctl status patch-agent
   sudo systemctl status patch-agent-heartbeat

TROUBLESHOOTING:
----------------
If agent services fail:
- Check logs: journalctl -u patch-agent -n 50
- Verify Python deps: /opt/patch-agent/venv/bin/python3 -c "import yaml, requests, flask"
- Check controller URL: cat /etc/patch-agent/env

If frontend doesn't load:
- Check nginx: systemctl status nginx
- Check backend: systemctl status patchmaster-backend
- Check logs: journalctl -u patchmaster-backend -n 50
