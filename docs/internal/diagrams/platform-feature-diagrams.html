12 Operator Instructions

Internal Build
  1. Run bash scripts/build-stable.sh.
  2. Keep dist/private/patchmaster-license-authority.env private.
  3. Never share dist/private/ with customers.
PatchMaster Server Install
  1. Copy dist/patchmaster-product-2.0.0.tar.gz to server.
  2. Extract and run bash packaging/install-bare.sh --with-monitoring.
  3. Installs with bundled runtime license bundle.
Vendor Server Install
  1. Copy vendor/dist/patchmaster-vendor-2.0.0.tar.gz to server.
  2. Extract and run bash install-vendor.sh.
  3. Installs with private signing authority.
After Both Installs
  1. Issue testing license first for POC hardware binding.
  2. Record hardware/MAC ID in Vendor portal.
  3. Generate final PM2-... license from Vendor tool.
  4. Paste license key into PatchMaster.
