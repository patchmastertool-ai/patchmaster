## Tier Descriptions

### **Basic**
Entry-level tier for small labs or personal use. Provides essential patching and snapshot capabilities for up to 10 hosts.

### **Basic DevOps**
Adds automation and observability tools for small dev teams. Includes software management, monitoring integration, and CI/CD hooks.

### **Standard**
The baseline for businesses. Adds critical compliance features like CVE tracking, Reporting, Audit Logs, and RBAC for teams managing up to 100 servers.

### **Standard DevOps**
Combines the business compliance features of Standard with the automation tools of DevOps. Perfect for modern tech companies.

### **Enterprise**
No limits. Full feature set, unlimited hosts, priority support, and advanced security features suitable for large-scale deployments.
